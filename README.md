# 动漫视频管理系统 java70

## 项目概述

本项目是基于 PHP 和 MySQL 开发的一套动漫视频管理系统，涵盖前台用户模块和后台管理员模块，支持登录、注册、视频播放、增删改查等功能。

## 技术栈

- 后端语言：PHP
- 数据库：MySQL
- 集成环境：phpstudy（推荐）、Wampsever、Xampp

## 功能模块

### 前台用户模块

- 用户注册与登录
- 视频信息查看与搜索
- 视频详情页播放、点赞与评论
- 个人中心资料修改与密码更改

### 后台管理员模块

- 管理员登录
- 视频信息增加、删除与修改
- 用户评论信息删除

## 系统角色

- 前台用户：可注册、登录、浏览视频、评论视频等
- 后台管理员：可登录、管理视频信息、删除用户评论等

## 运行环境

- PHP版本在5.6及以上

## 部署步骤

1. 准备运行环境，如安装 phpstudy、Wampsever 或 Xampp
2. 导入数据库文件，建立 9 张表
3. 部署代码到指定目录
4. 配置网站，确保 PHP 版本符合要求
5. 访问网站前台和后台，进行测试
